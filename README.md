# 🎧 Playwright - Spotify Clone Tests

Automação de testes E2E utilizando Playwright com TypeScript, simulando o comportamento de reprodução de músicas em um clone do Spotify.

## 🚀 Tecnologias utilizadas

* Playwright
* TypeScript
* Node.js

## 🧪 Cenários automatizados

* Deve reproduzir uma música
* Validação de usuário logado
* Mock de API de músicas (sem dependência de backend)

## 🧠 Diferenciais do projeto

* Uso de `page.route` para mock de API
* Testes desacoplados do backend
* Estrutura limpa e escalável

## ▶️ Como executar o projeto

```bash
npm install
npx playwright test
```

## 📊 Relatório de testes

```bash
npx playwright show-report
```
